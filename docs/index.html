StreamLoom

Weave Powerful Pipelines

StreamLoom is a framework designed to craft reusable blocks and compose them into efficient pipelines. With the flexibility to handle atomic and series data, various execution methods, and multiple data transports, StreamLoom offers versatility and power in pipeline construction.

Features

  • Easy creation of reusable blocks
  • Support for both atomic and series data types
  • Multiple execution methods: ProcessExecutor, AsyncExecutor, and more
  • Versatile data transport options: Sockets, Kafka queue, files, etc.
